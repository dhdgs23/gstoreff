# **App Name**: Garena Gears

## Core Features:

- Persistent Header: Display a prominent header with the website icon and navigation links (Home, Privacy Policy, T&C, Contact, About).
- Dynamic Image Slider: Implement an image slider showcasing attractive game-related visuals and offers.
- Discounted Item Display: Present product cards with Free Fire item images and discounted prices. The prices are set to 20, 40, 60, ... up to 240 for 12 cards.
- Purchase Initiation: Include a functional 'Buy' button on each product card.
- Informational Pages: Create dedicated pages for About, Privacy Policy, Terms & Conditions, and Contact, populated with relevant details.
- AI-Powered Customer FAQs: Utilize a generative AI tool to answer basic customer service questions via a simple chatbot (e.g. about delayed delivery of topups due to redeem codes).
- Concise Footer: Design an informative footer with essential links and copyright information.

## Style Guidelines:

- Primary color: Vivid reddish-orange (#F2542D) to reflect energy, excitement, and gaming spirit.
- Background color: Desaturated reddish-orange (#F9E9E5). Provides a light, unobtrusive backdrop.
- Accent color: Yellow-orange (#F2B84B). Used for interactive elements like buttons, adds contrast and calls attention to important actions.
- Font pairing: 'Poppins' (sans-serif) for headlines and shorter text snippets, and 'PT Sans' (sans-serif) for longer body text to ensure readability. Note: currently only Google Fonts are supported.
- Use icons related to gaming and the Free Fire universe to enhance the visual appeal of the site.
- Maintain a clean and professional layout with clear divisions and sections for easy navigation.
- Incorporate subtle animations and transitions for a smoother user experience when interacting with product cards and the image slider.
